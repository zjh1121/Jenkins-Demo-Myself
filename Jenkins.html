页面功能

这个页面包含布局、卡片、动画、主题切换和简单 JavaScript 效果

🚀

自动部署

Jenkins 构建成功后，会自动通过 SSH 把页面文件上传到服务器指定目录。

🔧

Pipeline 管理

所有流程都写在 Jenkinsfile 中，方便维护、修改和版本管理。

🌐

Nginx 访问

页面最终部署到 Nginx 网站目录，可以通过域名进行访问。
